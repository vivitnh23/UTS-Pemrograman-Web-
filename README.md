## Aplikasi Pemesanan Buku (UTS Pemrograman Web 1)
Repositori ini berisi project aplikasi web sederhana untuk pemesanan buku pada toko online.
Project dibuat untuk memenuhi tugas UTS mata kuliah Pemrograman Web 1, dengan fokus pada penggunaan HTML, CSS, dan JavaScript (tanpa backend).

## Fitur Aplikasi
Halaman	Deskripsi

Login	Validasi email & password, alert jika salah, modal pop-up untuk Lupa Password & Daftar.
Dashboard	Menu navigasi ke halaman lain dan greeting otomatis sesuai waktu.
Stok / Katalog	Menampilkan data buku secara dinamis dari data.js + fitur tambah stok menggunakan JavaScript DOM.
Checkout / Pemesanan	Form data pemesan dengan validasi input.
Tracking Pengiriman	Pencarian status pengiriman berdasarkan nomor order & menampilkan detail pengiriman.

---

## Teknologi yang Digunakan

HTML → Struktur halaman

CSS → Desain tampilan (external stylesheet)

JavaScript → Manipulasi DOM, validasi form, dan interaksi UI

Data dalam format Array JSON pada file data.js

Menggunakan JavaScript untuk:
